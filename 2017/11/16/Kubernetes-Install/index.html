Installing Docker

K8s(即’Kebunetes’) 1.8版本, 官方网站要安装说明安装了docker17.03, 若是docker版本过高, 也会有docker版本过高的提示, 所以这里就按照官网要求, 安装docker 17.03.

Kebernetes Dashboard

安装k8s dashboard github 地址 kubernetes/dashboard

然后执行kubectl -n kube-system edit service kubernetes-dashboard 将spec.type改为NodePort, 添加spec.porte.nodePort=31700, 31700为访问dashboard的IP.

之后浏览器访问https://:31700就可以查看dashboard.
